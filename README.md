# rubik

#### description:
rubik's cube simulator

#### building:
```sh
apt install libsdl2-dev
make
```

#### usage:
`./visualizer "U' L' U' F' R2 B' R F U B2 U B' L U' F U R F'"`

#### controls:
f, b, u, d, r, l: rotate corresponding face clockwise

**key modifiers**:
- left ctrl: rotate anti-clockwise
- left alt: rotate twice
